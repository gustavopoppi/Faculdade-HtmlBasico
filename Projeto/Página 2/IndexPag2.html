2º Trabalho Desenvolvimento de Aplicações


Trabalho para Nota parcial da primeira AEP da matéria de desenvolvimento de aplicações do professor Maurílio no 2º Semestre, onde dessa vez é necessário a utilização do HTML5.

Requisitos

AEP – 03

Desenvolvimento de uma página Web

O conteúdo da AEP – 03 será ministrado nas aulas até o dia 07/11/2019, sendo que será necessário desenvolver uma nova página principal 2, sendo adicionado um link para a página principal 2 na página principal das AEPs anteriores. Além disto, novas versões das seguintes páginas deverão ser desenvolvidas:

  • Música
  • Filmes
  • Séries
  • Games
  • Esportes

Sendo que nestas páginas deverão conter os conceitos de HTML5 e CSS3 vistos em sala.

As tags HTML5 que deverão ser utilizadas são:

  • article
  • aside
  • command
  • details
  • summary
  • figure
  • figcaption
  • footer
  • header
  • mark
  • nay
  • progress
  • section
  • time

A estilização via CSS/CSS3 deverá conter os seguintes itens:

  • Menu com dropdown (no topo ou lateral ou em alguma posição fixa da página)
  • Menu em todas as páginas com mesmo estilo

Para a AEP – 03 deveram ser entregues as páginas de:

  • Filmes
  • Séries

Deverão ser utilizados as tags vistas na aula do dia 07/10/2019 (header, nav, section, article, aside, footer) deixando o conteúdo das páginas de Filmes e Séries com tags do HTML5;

No header deverá ser utilizado informação textual e uma imagem;

Em nav deverá ser utilizado um menu drop-down, com informações referentes às páginas;

A tag section deverá ser composta de pelo menos dois artigos (tag article), sendo também utilizado a tag aside para referenciar o conteúdo interno da página (links internos);

No footer deverá ser utilizado uma imagem e informações sobre o desenvolvedor da página (nome, RA e turma).

NOVAS INFORMAÇÕES SERÃO ADICIONADAS A ESTE ARQUIVO PARA SEREM ENTREGUES NA AEP – 03.
